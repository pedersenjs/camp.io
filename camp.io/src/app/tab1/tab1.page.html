Camp.io

Camp.io is a web application which tracks campsites around the United States! On this main page you can search up campsites near you and plan your camping trip using the site details. Click on the marker to show information for the campsite! Don't see the campsite you're looking for? Add it to our map using the menu, and going to Add Location. All you need to add a location is the name, coordinates, and any other useful information you have that might make camping there easier! Got a bunch of campsites you like? Add them a list by going to the List menu. Thank you for using and supporting Camp.io!
